Prompts · Tu empresa en un documento
01 "Soy [nombre], tengo una empresa de [sector] que ofrece [servicio/producto] a [cliente ideal]. Ayúdame a escribir un documento Markdown que defina mi empresa: qué hacemos, a quién, propuesta de valor y tono de comunicación."
02 "A partir de este texto sobre mi empresa, extrae y estructura: misión, visión, público objetivo, propuesta de valor única, tono de marca y 3 mensajes clave. Formato Markdown."
03 "Revisa este documento de empresa y mejora la propuesta de valor. Que sea clara, directa y diferenciadora. Sin jerga. Máximo 2 frases."

PROFESIONAL
gamma.app
Bloque 2

Desde tu .md de empresa, creas una presentación real: elevator pitch, ponencia, deck para inversores. Prompteamos, iteramos, adaptamos tono.

Prompts · Gamma
01 "Crea una presentación de 8 slides para presentar mi empresa a clientes. Usa el documento adjunto como base. Tono profesional pero cercano."
02 "Genera un elevator pitch visual de 5 slides a partir de este .md. El público son inversores. Quiero que la primera slide impacte."
03 "Adapta esta presentación para una ponencia de 20 minutos. Añade más contexto en cada slide y una slide de preguntas al final."

Bloque 3

Con tu .md como briefing, generamos una landing page funcional. Web corporativa, página de producto, campaña. Sales con una URL publicada.

Prompts · Lovable / Bolt
01 "Crea una landing page para mi empresa. Usa este documento como briefing. Quiero hero, servicios, sobre nosotros y formulario de contacto."
02 "Genera una página de producto para [producto]. El CTA principal es 'Solicitar presupuesto'. Diseño limpio, sin imágenes por ahora."
03 "Crea una web de una sola página para mi consultoría. Incluye servicios, testimonios (ficticios de momento) y datos de contacto."
